Index of all calculators and how to use them

Welcome to the Open Calculator API.
Base URL: http://localhost:3000

1. BMI Calculator

[View Source Code: bmi.js]

Calculates Body Mass Index and health category.

  • Route: /api/bmi/calculate
  • Method: POST

Required JSON:

{
  "weight": 70,
  "height": 1.75
}

2. Basic Math Calculator

[View Source Code: math.js]

Simple arithmetic (add, subtract, multiply, divide).

  • Route: /api/math/calculate
  • Method: POST

Required JSON:

{
  "num1": 50,
  "num2": 10,
  "operation": "multiply"
}

3. GDP Calculator

[View Source Code: gdp.js]

Gross Domestic Product (Expenditure Approach).

  • Route: /api/gdp/calculate
  • Method: POST

Required JSON:

{
  "consumption": 1000,
  "investment": 500,
  "government": 400,
  "exports": 200,
  "imports": 300
}

4. Discount Calculator

[View Source Code: discount.js]

Calculate savings and final price.

  • Route: /api/discount/calculate
  • Method: POST

Required JSON:

{
  "originalPrice": 100,
  "discountPercent": 20
}

5. Simple Interest Calculator

[View Source Code: interest.js]

Calculate interest earned over time.

  • Route: /api/interest/calculate
  • Method: POST

Required JSON:

{
  "principal": 1000,
  "rate": 5,
  "time": 2
}

6. Temperature Converter

[View Source Code: temperature.js]

Convert Celsius to/from Fahrenheit.

  • Route: /api/temperature/calculate
  • Method: POST

Required JSON:

{
  "value": 30,
  "from": "celsius",
  "to": "fahrenheit"
}

7. Scientific Calculator

[View Source Code: scientific.js]

Advanced math functions (sin, cos, log, sqrt, pow).

  • Route: /api/scientific/calculate

Required JSON:

{
  "num": 90,
  "operation": "sin"
  // Optional: "power": 2 (if operation is 'pow')
}

8. Bill Splitter

[View Source Code: bill.js]

Split a restaurant bill with friends.

  • Route: /api/bill/calculate

Required JSON:

{
  "total": 150.00,
  "tipPercent": 15,
  "people": 4
}

9. Unit Converter

[View Source Code: units.js]

Convert common units (Length, Mass).

  • Route: /api/units/calculate

Required JSON:

{
  "value": 100,
  "type": "meters_to_feet"
}
