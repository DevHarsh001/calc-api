Index of all calculators and how to use them

Welcome to the Open Calculator API. This API is free for everyone to use for various calculations.

Important Info

  • Base URL (Local): http://localhost:3000
  • Base URL (Production): (You will replace this later when you host the backend)
  • All requests must use the header: Content-Type: application/json

1. BMI Calculator

Calculates Body Mass Index and provides a health category.

  • Route: /api/bmi/calculate
  • Method: POST

Required JSON Body:

You must send weight in kilograms (kg) and height in meters (m).

{
  "weight": 70,
  "height": 1.75
}

2. Basic Math Calculator

Performs simple arithmetic operations (add, subtract, multiply, divide).

  • Route: /api/math/calculate
  • Method: POST

Required JSON Body:

Provide two numbers and the operation you want to perform.

Valid operations: "add", "subtract", "multiply", "divide"

{
  "num1": 50,
  "num2": 10,
  "operation": "multiply"
}
